# add-1-to-a-number
Adds 1 to a number.

## Install
```
npm install add-1-to-a-number --save
```

## Usage
```javscript
var add1 = require('add-1-to-a-number');
console.log(add1(1));
// outputs: 2
```

### add1(number)
#### Number
Type: `number`
A number

#### Returns
A number OR throws `The 'number' given was not a number` if the number you give it is not actually a number.
